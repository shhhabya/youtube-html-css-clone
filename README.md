# YouTube Homepage Clone (HTML & CSS)

A responsive YouTube homepage user interface clone built using HTML5 and CSS3.

## Features
- Responsive video grid layout using CSS Grid
- Fixed header with search bar and tooltips
- Adaptive sidebar for different screen sizes
- Video cards with thumbnails, duration overlay, and channel details
- Interactive hover effects
- Clean and modular CSS structure

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)

## Responsive Design
- Optimized for mobile, tablet, and desktop screens
